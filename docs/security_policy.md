# 만파식(MPS) 보안 및 규제 준수 프레임워크

## 1. 준수 표준
- **FDA 21 CFR Part 11**: 전자 기록 및 전자 서명 규정 준수.
- **ISO 13485**: 의료기기 품질 경영 시스템 준수.
- **GDPR / 개인정보보호법**: 글로벌 개인정보 보호 가이드라인 준수.

## 2. 데이터 보안 아키텍처
- **암호화**: 
    - 저장 데이터 (Data at Rest): AES-256-GCM.
    - 전송 데이터 (Data in Transit): TLS 1.3.
- **무결성**: 블록체인 기반 해시 체인(Hash Chain)을 통한 데이터 변조 방지.
- **인증**: OAuth 2.0 + OpenID Connect (OIDC) 및 생체 인증 연동.

## 3. 감사 및 모니터링
- **Audit Logs**: 모든 데이터 접근 및 수정 이력을 해시 체인과 함께 기록.
- **자가 진단**: AI 자가치유 시스템을 통한 보안 취약점 실시간 탐지 및 대응.

## 4. 의료 데이터 관리
- **비식별화**: 연구용 데이터 공유 시 개인 식별 정보(PII) 자동 제거.
- **접근 제어**: 역할 기반 접근 제어(RBAC) 및 속성 기반 접근 제어(ABAC) 적용.
